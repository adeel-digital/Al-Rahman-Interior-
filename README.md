## 🚀 Getting Started

- **Live Website**: [adeel-digital.github.io/Al-Rahman-Interior-](https://adeel-digital.github.io/Al-Rahman-Interior-/)

### Local Development

1. **Clone the repository**
```bash
git clone https://github.com/adeel-digital/Al-Rahman-Interior-.git
cd Al-Rahman-Interior-
```

2. **Open in browser**
- Simply open `index.html` in your web browser
- Or use a local server:

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (if http-server installed)
http-server
```

3. **Access locally**
```
http://localhost:8000
```

## 🎯 Customization

### Change Logo
1. Replace `assets/logo.png` with your logo
2. Recommended size: 50x50px

### Modify Colors
Edit CSS variables in `css/style.css`:
```css
:root {
    --primary: #1a1a2e;      /* Dark navy blue */
    --secondary: #d4af37;    /* Gold accent */
    --accent: #16213e;       /* Darker navy */
    --light: #f5f5f5;        /* Light gray */
    --white: #ffffff;        /* White */
}
```

### Update Contact Information
Edit in `index.html` - Contact Section:
```html
<p>+1 (555) 123-4567</p>        <!-- Phone -->
<p>info@alrahmaninterior.com</p>  <!-- Email -->
```

### Add Portfolio Items
```html
<div class="portfolio-item" data-category="residential">
    <div class="portfolio-image">
        <div class="image-placeholder">
            <i class="fas fa-image"></i>
        </div>
    </div>
    <div class="portfolio-info">
        <h3>Project Name</h3>
        <p>Project description</p>
    </div>
</div>
```
